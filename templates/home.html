About This Demo

This demo showcases the CoreTablePage pattern, which provides:

  • Declarative Configuration - Define tables and filters with simple Python dictionaries
  • Dynamic Class Generation - Automatically creates django-tables2 Table and django-filter FilterSet classes
  • Bootstrap 5 Integration - Beautiful, responsive UI with crispy-forms
  • Minimal Boilerplate - Create complex data tables with just a few lines of code
  • Extensible Design - Easy to customize and extend for specific needs
